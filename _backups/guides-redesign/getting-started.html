Getting Started 5 min read Updated May 2026

This guide walks you through creating your first bithost account and deploying a cloud VPS paid entirely in Bitcoin. No ID, no credit card — just an email address and some crypto.

Prerequisites

  • A Bitcoin wallet (or any supported coin: ETH, LTC, DOGE, Lightning, etc.)
  • An SSH key pair (we'll create one if you don't have one)
  • 5 minutes

1. Create a free account

Go to dashboard.bithost.io/registration and enter your email address and a password. That's the entire signup flow — no phone number, no ID, no credit card.

You will receive a confirmation email. Click the link to verify your address (required to receive low-balance alerts).

2. Add an SSH key

bithost uses SSH key authentication — there are no passwords emailed to you. Before you can deploy a server, you need to add at least one public key.

Generate a key pair (skip if you already have one):

ssh-keygen -t ed25519 -C "your@email.com"

This creates ~/.ssh/id_ed25519 (private) and ~/.ssh/id_ed25519.pub (public). Copy the public key:

cat ~/.ssh/id_ed25519.pub

In the dashboard, go to Settings → SSH Keys → Add Key, paste the public key and give it a name.

3. Top up your balance

Click Add Funds in the dashboard. Choose your preferred cryptocurrency:

  • Bitcoin (BTC) — On-chain. Credits once the transaction has 1 confirmation (typically 10–20 min).
  • Lightning Network — Instant. Credits in under a second. Minimum deposit: $1.
  • ETH, LTC, DOGE, USDT, XRP, ZEC, BCH — Credits after network confirmation.

A fresh deposit address is generated for each top-up. Send any amount — your balance is denominated in USD and converted at the time of deposit.

4. Deploy your server

Click New Server. Configure:

  1. Provider — DigitalOcean, Linode, or Vultr (see the provider comparison guide).
  2. Region — Pick the datacenter closest to your users.
  3. Plan — Choose RAM, CPU, and disk. Plans start from $0.007/hr.
  4. OS image — Ubuntu, Debian, CentOS, Fedora, Windows, and more.
  5. SSH key — Select the key you added in step 2.

Click Deploy. Your server will be ready in 30–90 seconds.

5. Connect via SSH

Once the server status shows Active, copy the IP address from the dashboard and connect:

ssh root@YOUR_SERVER_IP

You're in. Your server is live, billed per hour, and paid entirely in crypto.

Next steps

  • Install the bh CLI — manage servers from your terminal without opening a browser.
  • Top up with Lightning — instant credits, no waiting for block confirmations.
  • Use the REST API — automate provisioning from your scripts and CI pipelines.
